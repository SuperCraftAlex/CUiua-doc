## Behaviour
<a anchor="out">Outputs</a> the first <a anchor="arg_count">n</a>
elements in the given <a anchor="arg_array">array</a>.

## Examples
```Python
[1 2 3 4] 2 ↙
# outputs: [1 2]
```
